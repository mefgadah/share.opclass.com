Stable Diffusion Web UI 安卓客户端

diffusion-client是一款强大的 Stable Diffusion Web UI 安卓客户端，AI 图像生成软件，支持文生图、图生图、模型管理、图片反推、reactor (换脸插件)、区域提示词插件等等，使用需要在电脑端安装服务器端手机才可以使用，需要一定的动手能力，非开箱即用。
